# [lute.dev](https://lute.dev)

Personal portfolio site for Matt Luchette, built with [Astro](https://astro.build/) and [React](https://reactjs.org/). 

## ✨ Project Features

- **Liquid Canvas**: An interactive WebGL fluid simulation at `/fluid`.
- **Design**: Built with a sapphire-themed design system, featuring a dotted grid and subtle ambient animations.
- **Career Highlights**: Dedicated sections for work history at Google, Intuit, and Oracle.
- **Tech Stack**: Astro 5, Tailwind CSS v4, and Vite.

## 🚀 Getting Started

```bash
# Install dependencies
pnpm install

# Start the dev server
pnpm run dev

# Build for production
pnpm run build
```
